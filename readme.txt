This plugin adds a small progress bar to the top of every wordress post page. 

== Description ==

Visit any post or page on your site and see a progress bar as you scroll. It completes when you reach the bottom of a post.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
